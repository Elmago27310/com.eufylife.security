Installatie instructies

Vanwege de manier waarop de Eufy Security API werkt, kan een combinatie van e-mail en wachtwoord niet
werken met zowel de Eufy Security mobiele app als de Homey app. 

Het wordt aanbevolen om de mobiele Eufy app te gebruiken om een extra "admin"-account te maken met een apart e-mailadres
en deze te gebruiken met deze app.

Gebruik
- Installeer deze app op Homey.
- Ga naar `apparaten toevoegen` 
- Voer gebruikersnaam en wachtwoord in. Klik op Volgende
- Beschikbare apparaten komen naar voren
